# Mpvpaper-client
Is a simple gui client written in python to easily manage mpvpaper
Note: please create a specific folder for this program to ensure the best functioning of the app

## Requirements
1.mpvpaper => https://github.com/GhostNaN/mpvpaper

2.```pip install requirements.txt```

## Run
1.```cd mpvpaper-client```

2.```python main.py```
